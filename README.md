# Vulnerability Scanner

A Python-based vulnerability scanner that detects open ports and identifies basic security risks.

## Features
- Port scanning
- Basic vulnerability checks
- Scan report generation

## Requirements
- Python 3.x

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python scanner.py
```

## Sample Output
```bash
Open ports found:
22 SSH
80 HTTP
443 HTTPS

Risk: Weak service exposure detected
```
